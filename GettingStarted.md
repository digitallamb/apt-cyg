# Introduction #

This page provides quick help to get you started improving apt-cyg.

### Get a working copy: ###
```
  # mkdir -p ~/src
  # svn checkout http://apt-cyg.googlecode.com/svn/trunk/ ~/src
  # chmod +x ~/src/apt-cyg
```

### Test the working copy: ###
```
  # ~/src/apt-cyg install nano
```
